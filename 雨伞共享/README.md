# 共享雨伞小程序

一个完整的共享雨伞解决方案，包含用户端小程序、管理端系统和后端服务。

## 项目结构

```
雨伞共享/
├── backend/          # 后端服务
├── mini-program/     # 用户端小程序
└── admin/            # 管理端系统
```

## 功能模块

### 用户端
- 微信授权登录
- 首页地图（定位、附近伞点、导航）
- 借伞（扫码、押金/免押、计费）
- 还伞（扫码、结算、退押金）
- 个人中心（订单、报修、押金）

### 管理端
- 点位管理
- 设备管理
- 订单管理
- 数据统计

## 技术栈

- 后端: Node.js + Express + MongoDB
- 用户端: 微信小程序原生开发
- 管理端: Vue.js + Element UI
